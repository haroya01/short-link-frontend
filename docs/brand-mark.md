# kurl 브랜드 마크 (3-bar) — 크로스 플랫폼 스펙 (SSOT)

세 개의 둥근 막대로 된 kurl 마크의 **정확한 비율·색·모션**. 모바일(iOS)·웹·앱·OG 카드·
파비콘이 전부 같은 형상을 쓰도록 하는 단일 기준. 새 표면을 만들 때 여기 좌표를 그대로 옮긴다.

## 1. 지오메트리 (불변)

`viewBox="0 0 28 18"` 안에 둥근 사각형 3개. **모든 막대: `height=3.4`, `rx=1.7`(=반높이라 양끝이 완전 반원).**

| 막대 | x | y | width | 비고 |
|---|---|---|---|---|
| 1 (위) | 6 | 1 | 20 | 오른쪽 정렬 느낌 |
| 2 (가운데) | 0 | 7.3 | 28 | 전폭 — 가장 길다 |
| 3 (아래) | 9 | 13.6 | 17 | 가장 짧다 |

- 막대 간 수직 간격(중심-중심) = 6.3 (1+1.7 → 7.3 → 13.6). 균일.
- 콘텐츠 세로 범위 = y 1.0 ~ 17.0 (높이 16). viewBox 18 이면 위아래 0.5px 숨. **iOS `KurlMark` 는 현재 28×17 프레임(84×51)을 쓴다 — 1px 캔버스 차이, 형상은 동일.** 신규/리팩터 시 28×18 로 맞추는 걸 권장(아래쪽 0.5px 숨이 잘림 방지).
- SVG 원본:
  ```svg
  <svg viewBox="0 0 28 18" xmlns="http://www.w3.org/2000/svg">
    <rect x="6" y="1"    width="20" height="3.4" rx="1.7"/>
    <rect x="0" y="7.3"  width="28" height="3.4" rx="1.7"/>
    <rect x="9" y="13.6" width="17" height="3.4" rx="1.7"/>
  </svg>
  ```

## 2. 색

- **기본(종이 위)**: 솔리드 브랜드 그린 `#059669` (= 웹 `accent-600`, iOS `Palette.accent`).
- **승격/다크/유리 위**: 세로 그린 그라데이션. 두 변형이 통용 중 — 신규는 밝은 쪽으로 통일 권장:
  - 밝은 그라데이션(OG·스플래시): `#34D399` → `#059669` (top-left→bottom-right).
  - 어두운 그라데이션(일부 OG 마크): `#10B981` → `#047857`.
- **단색 반전**: 그린 면/사진 위에서는 흰색(`#FFFFFF`). 그린 위 그린 금지.
- 브랜드 그린은 `#059669` 단일 기준 — 다른 초록 쓰지 말 것.

## 3. 모션 — "사사삭" 스태거 드로우

마크가 등장할 때 막대가 **왼쪽에서 줄별로 그려진다**(스플래시·로고 호버·로딩 로더 공통 언어):

- 줄별 시작 스태거: 0s / 0.08s / 0.16s (위→아래).
- 각 줄 드로우: duration **0.24s**, easing **cubic-bezier(0.22, 1, 0.36, 1)**.
- (스플래시 한정) 0.46s 에 워드마크가 7px 아래에서 떠오름.
- **로딩 로더**(`KurlLoadingMark`)는 이 드로우를 루프: 90ms 간격 3줄 → 620ms 유지 → 180ms 페이드아웃 → 280ms 후 반복.
- `reduce-motion` 이면 전부 정지 상태로 즉시 그려진다(애니메이션 없음).

## 4. 플랫폼별 구현 (이 스펙을 따르는 곳)

| 표면 | 파일 | 비고 |
|---|---|---|
| 웹 로고 | `components/common/logo.tsx` (`Mark`) | viewBox 28×18, `animated` 시 줄별 draw |
| 웹 OG 카드 | `lib/og.tsx` (`OgMark`) | 그라데이션, id 고유 |
| iOS 마크 | `kurl/Features/SplashView.swift` (`KurlMark`) | 프레임 84×51(28×17) — 18 로 정렬 권장 |
| iOS 로딩 로더 | `kurl/Features/Components.swift` (`KurlLoadingMark`) | 드로우 루프 |
| 파비콘 / 앱 아이콘 | `app/icon.svg`, `Assets.xcassets/AppIcon.appiconset` | 같은 막대 비율 유지 |

## 5. 여백·정렬 규칙

- 워드마크 옆에 둘 때: 마크 높이 = 워드마크 cap-height 와 맞추고, 간격 = 마크 폭의 ~0.3.
- 단독 아이콘(앱/파비콘): 마크를 캔버스의 ~52% 폭으로 중앙. 앱 아이콘은 둥근 사각형 배경(라이트=흰/페이퍼, 다크=잉크 네이비 `#05071e`, tinted=시스템).
- 최소 크기: 막대가 1.7px rx 라 16px 이하에선 둥글기가 뭉개짐 — 그 이하는 단색 블록으로 폴백 가능.
